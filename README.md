# SMHome_UniversalRound_ESP32_C3_Mini
PCB designed board for smart home curtain controll, garage doors or any other equipment with two direction motor
with max power 220v 10A.

Board has ESP-8266 (ESP1M form factor)  two relays and  poweer convertor& 

Main goal of this pcb board is to place  it under wall mounted electric components in socket box, 
but of cource it steel can bee plcad in its own box.


## Haw to use

```
git clone https://github.com/abel-msk/SMHome_2RelayBoardRound_ESP1M
cd SMHome_UniversalRound_ESP32_C3_Mini
```
Next step is to open project file  `SMHome_2RelayBoardRound_ESP1M_v7.kicad_pro`  in KiKad. 

All footprint aggregated in local library `Library.pretty`, shematic symbols in `Library.kicad_sym`. You can add path to this library in project settings.
